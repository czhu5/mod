ABOUT

Mastermind of Data (MOD) will provide the ability to edit databases without deleting previously existing data and eliminate duplicate information by merging data from multiple entries for simplification within a database. Our product has the capability to automatically execute commands to edit data entries, delete duplicates, and merge data. To execute this, our product displays a menu bar that will allow the user to add, delete, merge, save, reload, scroll, filter, and search for data entries.

HOW IT WORKS

Access to Mastermind of Data is granted through the purchase of a license. Once the license has been obtained, the user will receive Login information which will grant access to the MOD website. Once logged in, the user can then import a database from Microsoft Access and begin to manipulate data in various ways such as, merging duplicate files, adding, deleting, and filtering records without the need to hire outside personnel.
